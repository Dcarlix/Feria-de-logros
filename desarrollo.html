¿QUÉ ES DESARROLLO DE SOFTWARE?


El desarrollo de software se refiere a un conjunto de actividades informáticas dedicadas al proceso de creación, diseño, despliegue y compatibilidad de software. En este caso podemos distinguir a tres tipos básicos: Software de sistema, Software de programación, Software de aplicación.



Cada uno de estos tipos tiene un propósito y una funcionalidad específica para abordar objetivos comerciales. El desarrollo de software lo llevan a cabo principalmente programadores, ingenieros de software y desarrolladores de software. Los pasos del proceso de desarrollo de software pueden variar según la metodología elegida, pero en general se pueden resumir en siete etapas: Planificación, Análisis, Diseño, Prototipado, Desarrollo, Pruebas, Implementación. Estos pasos permiten definir el alcance, los requisitos, las soluciones, el código, la calidad, el despliegue y el soporte del software.


¿Porque implementamos esto?


Pues en lo largo de nuestro año como estudiante de desarrollo hemos visto sobre estas metodologías, las hemos estudiando y aplicado, ya que hemos hecho documentos Word aplicando esa metodología y desarrollando la explicación y por qué son importantes. Para empezar en esta especialidad, aprendemos como primero sobre la metodología de solución de problemas, cada vez vamos haciendo más Actividades entrando a lo que ya son los códigos. En nuestra experiencia base de nuestro año como desarrollador hemos diseñado códigos y hecho pruebas y tener en cuenta sobre el proceso de requerimientos.



¿Qué se hace en Desarrollo de Software?

En el desarrollo de software se realizan un conjunto de actividades informáticas dedicadas al proceso de creación, diseño, despliegue y compatibilidad de software. Todas estas son realizadas a través de nuestra laptop, comenzaras con una parte de diseño luego de ello vas entrando más en la codificación en pocas palabras programar. En desarrollo de software realizaras diferentes proyectos tales como una calculadora, control de notas, un software para un cajero o hasta un videojuego creado desde tu computadora Algunos programas utilizados este año fueron: Pseint, Bloc de notas, Visual studio community, Notepad ++, Visual studio code, Entre otros…



Algunos lenguajes utilizados fueron: HTML5, CSS, JAVASCRIPT, C#, Entre otros… Además del área tecnológica se tienen las diferentes materias básicas tales como: Matemática, Sociales, Ingles, Ciencias y demás … En conclusión, la carrera de desarrollo de software se dedica mayormente a la realización de actividades tecnológicas para crear diferentes proyectos como fueron mencionados anteriormente




¿Qué hemos visto?

A lo largo del año, nosotros como desarrolladores hemos visto deferentes aplicaciones y lenguajes de programacion para aprender de este mundo, que como pueden observan en el fondo son algunos de los programas y lenguajes que hemos visto que son:


HTML5 (HyperText Markup Language 5): Es la última evolución del lenguaje HTML utilizado para crear estructura y contenido en sitios web. HTML5 introduce nuevas etiquetas y características que permiten una mejor representación de contenido multimedia y una mayor interactividad.


CSS3 (Cascading Style Sheets 3): CSS3 es la última versión de las hojas de estilo en cascada y se utiliza para controlar la presentación y el diseño de elementos HTML en una página web. Incluye nuevas características y mejoras en comparación con versiones anteriores de CSS.


JavaScript: Es un lenguaje de programación ampliamente utilizado en el desarrollo web para agregar interactividad y funcionalidad a las páginas web. Permite a los desarrolladores crear aplicaciones web dinámicas y controlar el comportamiento del cliente en el navegador.


C# (C Sharp): Es un lenguaje de programación desarrollado por Microsoft. Se utiliza en una variedad de aplicaciones, incluyendo desarrollo de software de escritorio, aplicaciones web y juegos.


PSeInt: Es un entorno de desarrollo y aprendizaje de pseudocódigo utilizado para enseñar los conceptos de programación y algoritmos. Está diseñado para principiantes en la programación.


Photoshop: Adobe Photoshop es una aplicación de edición de imágenes ampliamente utilizada que permite la manipulación y creación de gráficos, fotografías y diseño.


Macromedia Flash (anteriormente): Flash era una plataforma multimedia y de animación desarrollada por Macromedia y posteriormente adquirida por Adobe. Se utilizaba para crear animaciones interactivas y contenido multimedia en la web.


GIMP (GNU Image Manipulation Program): GIMP es una aplicación de edición de imágenes de código abierto y gratuita. Ofrece muchas de las mismas funciones que Photoshop y se utiliza para editar y crear imágenes.


Notepad++: Es un editor de texto avanzado y de código abierto para sistemas Windows. Es ampliamente utilizado por programadores para escribir y editar código fuente en varios lenguajes de programación.


Visual Studio Code: Es un editor de código fuente de código abierto desarrollado por Microsoft. Es altamente personalizable y se utiliza para escribir y depurar código en una variedad de lenguajes de programación.


Visual Studio Community: Es una versión gratuita de Microsoft Visual Studio que ofrece herramientas de desarrollo para una variedad de aplicaciones, incluyendo aplicaciones de escritorio, aplicaciones web y móviles.


Bloc de Notas (Notepad): Es un editor de texto simple que se incluye en los sistemas operativos Windows. Se utiliza para editar archivos de texto sin formato y a menudo se utiliza para escribir scripts o código pequeño.


Wondershare: Wondershare es una empresa que ofrece una variedad de software, incluyendo aplicaciones de edición de video, herramientas de recuperación de datos y utilidades multimedia.



¿Qué necesitas para ser desarrollador?

Durante el tiempo que nosotros hemos pasado estudiando esta carrera ya por un año, pues podemos decirles que para ser un desarrolador se necesitan de varias cualidades para que nos termine de gustar esta materia, muchos de estas cualiades son las siguientes 3:


1.Paciencia: Se necesita paciencia para poder entender a los diferentes temas de programacion, y también debemos ser pacientes a la hora de corregir los errores o problemas que nos puede dar el codigo


2.Curiosidad: Esta materia tiene mucho lo que es el ser una persona autodidacta, es decir, que todo lo que sabemos o la gran parte de lo que sabemos, lo hemos aprendido de una forma independiente a que alguien nos enseñe, ya que para poder entender completamente los temas uno debe informarse bien de cada cosa que nosotros no sabemos o de las cosas que son nuevas para nosotros, y tambien aprender nuevos lenguajes de programacion


3.Ser analítico: Debemos ver y entender por partes de manera sistematica y ordenada los problemas que se nos proponen de solucionar, y tambien entra el valor anterior, ya que tenemos que tener las ganas de aprender mas cosas y mas cosas para poder obtener los conocimeintos necesarios para poder hacer cualquier programa que haga un sin fin de funciones.


En conclusión, si les interesa el desarrollo de software, que no sea por la dicha fama de los sueldos, o el hecho de que todo se va modernizando, sino que es mejor guiarse por el hecho de que este tema les gusta, el poder crear programas, hacer funciones, resolver problemas de caracter digital. Esta carrera no es para todos, pero si les interesa y si les gusta, pues el dedicarse a ella es una buena decision, principalmente porque estan estudiando algo que les gusta.
